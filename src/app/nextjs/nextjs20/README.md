# Next.js 20: Создание Server Action для создания инвойса

## Задача
Создать Server Action для мутации данных - добавления нового инвойса в коллекцию.

## Инструкция
1. Найдите TODO комментарий в файле `actions.ts`
2. Реализуйте функцию `createInvoice` для обработки данных формы и сохранения инвойса

## Требования
- Добавить директиву `'use server'` в начало функции
- Извлечь данные из объекта FormData
- Валидировать данные с использованием Zod
- Преобразовать сумму из долларов в центы
- Добавить запись в коллекцию через функцию `addInvoice`
- Вызвать `revalidatePath` и `redirect` после мутации

## Подсказка
Для извлечения данных из FormData используйте метод `formData.get('имя_поля')`. Функции `revalidatePath` и `redirect` находятся в разных модулях: `next/cache` и `next/navigation`.
