# Next.js 03: Lazy Loading Images

## Задача
Настройте lazy loading для изображений, чтобы они загружались только при необходимости.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Измените значение пропса `loading` для компонентов `next/image`
3. Убедитесь, что изображения загружаются только при вхождении в viewport

## Требования
- Используйте lazy loading для изображений в галерее
- Первое изображение может загружаться сразу (eager)
- Остальные изображения должны использовать lazy loading
- Убедитесь, что изображения отображаются корректно

## Подсказка
Измените значение пропса `loading` с "eager" на "lazy" для второго изображения в галерее.
