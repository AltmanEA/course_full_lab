# Next.js 17: Базовый поиск с debounce

## Задача
Реализовать поиск товаров с обновлением URL и фильтрацией таблицы.

## Инструкция
1. Найдите TODO комментарий в файле `Search.tsx`
2. Реализуйте функцию `handleSearch` с использованием `router.push()` и `debounce`
3. В файле `page.tsx` передайте `searchParams` в компонент `ProductTable`

## Требования
- При вводе текста URL обновляется с параметром `?query=...`
- Обновление URL происходит через 300мс без перезагрузки страницы
- Таблица отображает только отфильтрованные данные

## Подсказка
Используйте `useRef` для хранения таймера debounce и очищайте его в `useEffect`.
