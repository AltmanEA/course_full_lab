# Next.js 19: Комбинированный поиск и пагинация

## Задача
Реализуйте совместную работу поиска и пагинации с использованием URL параметров. При изменении поиска страница должна сбрасываться на первую, а при навигации по страницам поисковый запрос должен сохраняться.

## Инструкция
1. Найдите TODO комментарий в файле `Search.tsx` и реализуйте функцию `updateURL`
2. Найдите TODO комментарий в файле `Pagination.tsx` и реализуйте функцию `createPageURL`
3. Найдите TODO комментарий в файле `page.tsx` и передайте оба параметра в запрос данных

## Требования
- Функция `updateURL` должна использовать URLSearchParams для управления параметрами
- При изменении поиска страница должна сбрасываться на 1
- URL должен содержать оба параметра: `?query=abc&page=2`
- При обновлении страницы состояние должно восстанавливаться

## Подсказка
Используйте `URLSearchParams` для безопасной работы с параметрами URL. Метод `set()` позволяет добавлять или изменять параметры, а метод `delete()` — удалять их.
