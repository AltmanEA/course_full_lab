# nextjs26 - Semantic HTML и Labelling в формах

## Задача

Форма создания инвойса содержит поля для выбора клиента, ввода суммы и статуса. Сейчас элементы формы свёрнуты в `div` без использования семантических элементов.

### Что нужно сделать

Заменить `div` на соответствующие семантические элементы формы и добавить связывание полей с подписями.

### Критерии проверки

- Поля должны быть свёрнуты в `<select>` и `<input>`
- Каждое поле должно иметь связанный `<label>` с атрибутом `for`, указывающим на `id` поля
