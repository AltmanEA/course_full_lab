# Next.js 08: Programmatic Navigation with useRouter

## Задача
Добавить программную навигацию на странице с формой. После отправки формы пользователь должен быть перенаправлен на другую страницу.

## Проблема
В исходном состоянии форма отправляется, но перехода на другую страницу не происходит. Пользователь остаётся на той же странице.

## Что нужно сделать
1. Импортировать `useRouter` из `next/navigation`
2. Создать обработчик отправки формы
3. Использовать `router.push()` для перенаправления на страницу успеха
4. Создать страницу success, куда будет происходить перенаправление

## Концепция Programmatic Navigation
`useRouter` hook из `next/navigation` позволяет:
- Программно переходить между страницами
- Использовать `router.push()` для навигации (как history.pushState)
- Использовать `router.replace()` для замены текущей записи в истории
- Передавать state между страницами
- Работать с query параметрами
