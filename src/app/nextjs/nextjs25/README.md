# nextjs25 - notFound() и not-found.tsx

## Что изучается

- Обработка 404 для несуществующих ресурсов
- Приоритет `notFound()` над `error.tsx`

## Задание

Страница `/dashboard/invoices/[id]/edit` загружается для любого ID, даже несуществующего — показывает пустую форму или падает.

### Что нужно сделать

1. В функции получения данных добавить проверку: если инвойс не найден — вызвать `notFound()`
2. Создать файл `not-found.tsx` в папке `invoices/[id]/edit` с кастомным UI для страницы 404

### Проверка

При переходе на несуществующий ID должна отображаться страница 404, а не ошибка приложения.
