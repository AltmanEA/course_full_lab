# Next.js 07: Active Link Navigation

## Задача
Реализуйте стилизацию активной ссылки в навигации, чтобы текущая страница была визуально выделена.

## Инструкция
1. Найдите TODO комментарий в файле `layout.tsx`
2. Добавьте стилизацию для активной ссылки, чтобы она отличалась от других ссылок

## Требования
- Активная ссылка должна визуально отличаться от других ссылок
- Стилизация должна применяться динамически в зависимости от текущей страницы
- Используйте `usePathname` хук для определения текущей страницы
- Активная ссылка должна иметь классы `border-indigo-500` и `text-gray-900`
- Неактивные ссылки должны иметь классы `border-transparent` и `text-gray-500`

## Подсказка
Используйте условный рендеринг классов в зависимости от того, совпадает ли href ссылки с текущим pathname.
