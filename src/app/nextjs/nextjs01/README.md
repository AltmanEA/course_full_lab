# Next.js 01: Optimizing Images

## Задача
Оптимизировать изображение на странице, заменив обычный HTML тег `<img>` на компонент `next/image` из Next.js.

## Проблема
В исходном состоянии страница использует обычный HTML тег `<img>` для отображения изображения. Это приводит к:
- Медленной загрузке изображения
- Отсутствию автоматической оптимизации
- Нет поддержки современных форматов изображений
- Отсутствию lazy loading по умолчанию

## Что нужно сделать
1. Заменить HTML тег `<img>` на компонент `next/image`
2. Настроить правильные размеры изображения (width, height)
3. Убедиться, что изображение оптимизируется автоматически
4. Проверить, что lazy loading работает корректно

## Концепция Image Optimization
Next.js предоставляет встроенную оптимизацию изображений через компонент `next/image`, который:
- Автоматически оптимизирует изображения в современных форматах (WebP, AVIF)
- Реализует lazy loading по умолчанию
- Уменьшает размер изображений на основе device size
- Поддерживает placeholder'ы во время загрузки
- Улучшает Core Web Vitals (LCP, CLS)
