# Next.js 01: Optimizing Images

## Задача
Замените обычный HTML тег `<img>` на компонент `next/image` для оптимизации изображений.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Замените тег `<img>` на компонент `next/image`
3. Добавьте необходимые пропсы width и height
4. Убедитесь, что изображение отображается корректно

## Требования
- Используйте компонент `next/image` вместо обычного тега `<img>`
- Укажите правильные значения для пропсов width и height
- Изображение должно отображаться без ошибок
- Сохраните все остальные стили и атрибуты

## Подсказка
Импортируйте компонент `Image` из 'next/image' и используйте пропсы width={500} и height={250} для задания размеров.
