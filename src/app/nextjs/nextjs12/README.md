# Next.js 12: Static Rendering with Caching

## Задача
Оптимизируйте загрузку данных, используя статический рендеринг с кэшированием.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Создайте асинхронную функцию `getData()` с fetch запросом
3. Добавьте опцию `{ cache: 'force-cache' }` к fetch
4. Вызовите функцию `getData()` в компоненте

## Требования
- Используйте `async/await` для получения данных
- API endpoint: `/api/nextjs12`
- Примените `{ cache: 'force-cache' }` к fetch запросу
- Данные должны кэшироваться и не загружаться повторно

## Подсказка
Используйте `fetch` с опцией `cache: 'force-cache'`, затем `await` для получения результата.
