# ARIA атрибуты для доступности форм

Добавьте ARIA атрибуты для улучшения доступности формы для пользователей программ чтения с экрана (скринридеров).

## Задача

Форма имеет валидацию email, но сообщения об ошибках недоступны для пользователей скринридеров.

## Что нужно сделать

1. Добавьте `aria-describedby="email-error"` к полю ввода email
2. Добавьте `id="email-error"` к контейнеру с сообщением об ошибке
3. Добавьте `aria-live="polite"` к контейнеру ошибок для анонса изменений
