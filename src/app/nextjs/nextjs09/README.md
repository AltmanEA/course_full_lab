# Next.js 09: Parallel Data Fetching

## Задача
Оптимизируйте загрузку данных, используя параллельный fetch вместо последовательного.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Оберните вызовы `getUsers()` и `getTasks()` в `Promise.all([...])`
3. Получите результаты через деструктуризацию

## Требования
- Используйте `Promise.all` для параллельной загрузки
- Деструктурируйте массив результатов
- Оба списка (users + tasks) должны отображаться

## Подсказка
`Promise.all([getUsers(), getTasks()])` запускает запросы одновременно.
