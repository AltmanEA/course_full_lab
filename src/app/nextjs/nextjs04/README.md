# Next.js 04: Creating Route Segments

## Задача
Создайте правильную структуру папок для нового route segment, чтобы страница была доступна по пути /nextjs/nextjs04/demo.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Создайте новую папку `demo` в директории задачи
3. Переместите файл `page.tsx` в папку `demo`
4. Убедитесь, что страница доступна по новому адресу

## Требования
- Страница должна быть доступна по адресу /nextjs/nextjs04/demo
- Исходный файл page.tsx должен находиться в папке demo
- Структура папок должна соответствовать маршруту
- Все ссылки на страницу должны вести на новый адрес

## Подсказка
Создайте папку `demo` и переместите в нее файл `page.tsx`. Это изменит маршрут страницы с /nextjs/nextjs04 на /nextjs/nextjs04/demo.
