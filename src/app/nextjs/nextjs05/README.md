# Next.js 05: Dashboard Nested Layout

## Задача
Создайте общий вложенный layout для dashboard страниц, чтобы избежать дублирования кода навигации и структуры.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Создайте файл `layout.tsx` в папке nextjs05
3. Перенесите общую структуру страницы в layout
4. Удалите дублирующийся код из отдельных страниц

## Требования
- Создайте общий layout для всех dashboard страниц
- Перенесите навигацию и основную структуру в layout
- Убедитесь, что все страницы используют общий layout
- Сохраните функциональность всех страниц

## Подсказка
Создайте файл `layout.tsx` который будет содержать общую структуру dashboard страниц. Перенесите в него навигацию и основной layout, оставив уникальный контент в отдельных страницах.
