# Next.js 02: Optimizing Fonts and Images

## Задача
Оптимизируйте шрифты и изображения на странице, используя компоненты next/font и next/image.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Импортируйте шрифт через `next/font/google`
3. Замените тег `<img>` на компонент `next/image`
4. Примените оптимизированный шрифт к текстовым элементам

## Требования
- Используйте компонент `next/image` для изображения
- Импортируйте шрифт через `next/font/google`
- Примените шрифт к заголовкам и текстовым элементам
- Укажите правильные размеры для изображения

## Подсказка
Импортируйте шрифт Inter из 'next/font/google' и используйте его для стилизации текста. Для изображения используйте пропсы width={500} и height={250}.
