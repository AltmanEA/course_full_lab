# Next.js 02: Optimizing Fonts and Images

## Задача
Оптимизировать шрифты и изображения на странице, заменив обычные CSS шрифты на компоненты next/font и убедиться, что изображения также оптимизированы.

## Проблема
В исходном состоянии страница использует:
- Обычные CSS шрифты через @import или link теги
- Стандартные HTML изображения
Это приводит к:
- Медленной загрузке шрифтов (FOUT - Flash of Unstyled Text)
- Отсутствию оптимизации шрифтов
- Нет автоматической оптимизации изображений
- Увеличенному размеру бандла

## Что нужно сделать
1. Импортировать шрифты через `next/font/google` или `next/font/local`
2. Заменить обычные изображения на `next/image`
3. Настроить правильные размеры для изображений
4. Убедиться, что шрифты загружаются оптимально
5. Проверить, что нет FOUC (Flash of Unstyled Content)

## Концепция Font и Image Optimization
Next.js предоставляет:
- **next/font**: Автоматически оптимизирует шрифты, self-hosting, предотвращает FOIT
- **next/image**: Автоматическая оптимизация изображений в современных форматах
- Оба компонента улучшают Core Web Vitals (LCP, CLS)
- Автоматический code splitting для шрифтов
