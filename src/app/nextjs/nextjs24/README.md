# Next.js 24: Error Boundary для границы ошибок

## Задача
Страница /dashboard/invoices выбрасывает ошибку при загрузке. Обработчика ошибок нет.

## Инструкция
1. Создайте файл `error.tsx` в папке `src/app/nextjs/nextjs24/invoices/`
2. Добавьте директиву `"use client"`
3. Создайте компонент с props `error` и `reset`
4. Выведите сообщение об ошибке из `error.message`
5. Добавьте кнопку с вызовом `reset()` для повторной попытки

## Требования
- Файл `error.tsx` должен быть в той же папке, что и `page.tsx`
- Компонент должен быть Client Component (`"use client"`)
- Должен отображать `error.message`
- Должен содержать кнопку вызова `reset()`

## Подсказка
Error boundary в Next.js автоматически перехватывает ошибки в сегменте маршрута. Компонент получает объект ошибки и функцию reset для повторного рендера.
