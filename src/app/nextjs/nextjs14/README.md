# Next.js 14: Basic Page Streaming

## Задача
Добавьте индикатор загрузки для улучшения UX при навигации на страницу с асинхронной загрузкой данных.

## Инструкция
1. Создайте файл `loading.tsx` в папке `src/app/nextjs/nextjs14/`
2. Добавьте простой fallback UI (текст "Загрузка..." или скелетон)
3. Файл loading.tsx автоматически создаст Suspense boundary для страницы

## Требования
- Создайте файл `loading.tsx` в той же папке, что и `page.tsx`
- Используйте любой визуальный индикатор загрузки
- При навигации должен отображаться loading state, а не пустой экран

## Подсказка
Next.js автоматически использует файл `loading.tsx` как Suspense fallback при загрузке страницы.
