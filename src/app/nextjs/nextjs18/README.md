# Пагинация с параметрами URL (Next.js 15)

## Описание

Реализуйте компонент пагинации, который управляет номером текущей страницы через URL параметры.

В Next.js 15 `searchParams` в серверных компонентах является Promise, поэтому требуется использование `await`.

## Что нужно сделать

В файле `Pagination.tsx` допишите:

1. Функцию `createPageURL(pageNumber)` для создания URL с параметром страницы
2. Обработчики `onClick` для кнопок "Предыдущая" и "Следующая"
3. Условие `disabled` для кнопок на границах диапазона

## Требования

- При клике на кнопку URL должен обновляться через `router.push()`
- Номер страницы читается из `searchParams`
- Кнопка "Предыдущая" недоступна на первой странице
- Кнопка "Следующая" недоступна на последней странице
- Отображается 5 элементов на странице
