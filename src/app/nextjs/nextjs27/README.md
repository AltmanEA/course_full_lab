# Серверная валидация с useActionState

## Цель
Научиться обрабатывать ошибки валидации формы на сервере с помощью хука `useActionState` и возвращать понятные сообщения пользователю.

## Задача
Форма создания инвойса не обрабатывает ошибки валидации. При отправке пустой формы возникает ошибка сервера вместо дружелюбных сообщений.

## Что нужно сделать
1. Добавить хук `useActionState` в компонент формы для получения состояния ошибок
2. Заменить `parse()` на `safeParse()` в Server Action
3. Добавить проверку результата валидации и возврат объекта ошибок

## Проверка
Тесты проверяют, что:
- При отправке пустой формы отображаются сообщения об ошибках для каждого поля
- При успешной отправке валидной формы показывается сообщение об успехе
- Частично заполненная форма показывает ошибки только для пустых полей
