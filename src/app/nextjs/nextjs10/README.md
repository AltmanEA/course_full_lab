# Next.js 10: Async Data Fetching in Server Component

## Задача
Преобразуйте статический компонент в асинхронный Server Component и загрузите данные с сервера.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Сделайте компонент асинхронным (`async function Home()`)
3. Импортируйте функцию `getUsers` из `../data`
4. Вызовите `await getUsers()` и сохраните результат
5. Отобразите полученные данные вместо статичного массива

## Требования
- Используйте `async/await` для получения данных
- Импортируйте функцию из файла `../data`
- Компонент должен быть асинхронным
- Отобразите реальные имена пользователей

## Подсказка
Добавьте `async` к определению функции и используйте `await getUsers()` для получения данных.
