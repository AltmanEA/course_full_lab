# Next.js 13: Dynamic Rendering for Real-Time Data

## Задача
Настройте динамический рендеринг для получения актуальных данных в реальном времени.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Создайте асинхронную функцию `getData()` с fetch запросом к `/api/nextjs12`
3. Добавьте опцию `{ cache: 'no-store' }` к fetch запросу
4. Вызовите функцию `getData()` и отобразите полученные данные

## Требования
- Используйте `async/await` для получения данных
- Отключите кэширование с помощью `{ cache: 'no-store' }`
- Данные должны обновляться при каждом обновлении страницы
- Отобразите `timestamp` из API ответа

## Подсказка
Используйте `fetch` с опцией `cache: 'no-store'` для получения свежих данных при каждом запросе.
