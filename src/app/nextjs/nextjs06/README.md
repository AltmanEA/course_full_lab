# Next.js 06: Basic Navigation with Link Component

## Задача
Замените стандартные HTML теги `<a>` на компонент `Link` из Next.js для реализации клиентской навигации между страницами.

## Инструкция
1. Найдите TODO комментарий в файле `page.tsx`
2. Импортируйте компонент `Link` из 'next/link'
3. Замените теги `<a>` на компоненты `Link`
4. Убедитесь, что навигация работает корректно

## Требования
- Используйте компонент `Link` вместо тегов `<a>`
- Сохраните все стили и атрибуты ссылок
- Навигация должна происходить без полной перезагрузки страницы
- Все ссылки должны вести на правильные страницы задач

## Подсказка
Импортируйте компонент `Link` из 'next/link' и оберните в него содержимое тегов `<a>`. Убедитесь, что пропс `href` передается компоненту `Link`.
